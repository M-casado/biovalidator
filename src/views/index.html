About this endpoint

This service uses Biovalidator to validate FEGA/EGA metadata JSON against JSON Schema. Paste a schema and a JSON document, or fetch the current minimal valid examples from the FEGA metadata schema repository and load one into the editor.

The fetched examples are request wrappers with top-level schema and data fields. The schema commonly points to the FEGA schema repository with $ref, while the data field contains the JSON metadata instance that will be validated.

More information is available in the Biovalidator repository, the FEGA metadata schema repository, and the FEGA metadata technical report.
